The Nine Billion Names

Inspired by Arthur C. Clarke's short story. Set the alphabet size, test whether the naming rules can reach nine billion names, then generate a sample of 100 distinct valid names.

1) Alphabet And Feasibility

Rules used for counting names: each name has length 1 to 9, and no character may appear more than 3 times in a row.

We count valid names by dynamic programming. Let f(L, r) be the number of length-L names whose last run has length r (r = 1, 2, 3).

Transitions for alphabet size n: f(L, 1) = (n - 1) x [f(L-1,1) + f(L-1,2) + f(L-1,3)], f(L, 2) = f(L-1,1), f(L, 3) = f(L-1,2).

Base case: f(1,1) = n, f(1,2)=0, f(1,3)=0. Total valid names = sum over L=1..9 of [f(L,1)+f(L,2)+f(L,3)].
